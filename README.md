# Red Black Tree Implementation


# Uputstvo
1. `git clone https://github.com/neske99/RedBlackTreeImplementation.git`
2. Make ima naredne komande: run,topdf,cleanDotFiles,clean i exec
3. Make run pravi izvrsni run fajl koji je kompajlirani test.cpp
4. Make topdf konvertuje .dot iz dots direktorijuma i smesta ih u pfds/dots folder
5. Make cleanDotFiles brise .dot fajlove iz dots direktorijuma
6. make clean brise .dot fajlove .pdf fajlove(iz odgovarajucih direktorijuma) i run file
